Cloud9配合ngrok搭建shadowsocks免费vpn实现科学上网

发表于 2016-04-23   |   分类于 分享   |  

😳好吧标题略长

###原理
使用cloud9提供的开发环境vps安装shadowsocks服务端，因为cloud9的vps没有提供公网ip，所以使用ngrok的tcp tunnel 进行内网穿透,穿透后用获取来的IP和端口配置shadowsocks客户端。

###准备工作

  • 申请cloud9账号一个(用github的同学直接登录就可以咯)
  • 申请ngrok帐号一个 (话说这个已经被墙了！那肿么办！😄先不急，往下看)
  • 添加一个开发环境（具体操作按照官网提示走就行）
  • 下载shadowsocks客户端
    ###安装shadowsocks服务端
  1. 打开cloud9的开发环境
  2. 左下角terminal输入sudo passwd root(修改root密码)
  3. 继续输入su root（需要使用root权限安装）
  4. apt-get update
  5. pip install shadowsocks
    ###配置并启动shadowsocks服务端
  6. 在任意目录编写一个配置文件 如（/etc/shadowsocks.conf），内容如
    下，(参数说明可参考koding搭建shadowsocks)

{

“server”:”0.0.0.0”,
“server_port”:8123,
“password”:”password”,
“timeout”:500,
“method”:”aes-256-cfb”,
“fast_open”: false,
“workers”: 10

}

2.启动shadowsocks 服务端

###配置ngrok
1.在cloud9的终端下载ngrok的linux版本客户端

2.解压

3.配置ngrok的authtoken(下面这个是我的token，大家翻墙后自己去ngrok注册换成自己的token再运行一下就可以了)

4.开启tcp tunnel 进行内网穿透(此处8123端口为第一步配置的shadowsocks服务端的端口)

连接后会出现一个0.tcp.ngrok.io:端口

###配置shadowsocks客户端

  1. 启动shadowsocks客户端，系统托盘会出现小飞机的图标
  2. 右键小飞机图表，选择服务器，点击编辑服务器
  3. 选择添加，将服务器端的信息填入（其中服务器ip通过 ping 0.tcp.ngrok.io获得）
  4. 最后勾选启动系统代理即可

    建议翻墙成功后更新PAC
    ###后话

    有空的童鞋可以多申请几个，shadowsocks最新的客户端支持负载均衡～你知道的！
